# Desktop Build Assets

Place the following assets here for packaging:
- `icon.icns` (macOS App Icon)
- `icon.ico` (Windows App Icon)
- `icon.png` (Linux App Icon, 512x512)
- `background.png` (DMG Installer background)

## Specifications
- **macOS**: 1024x1024 px minimum.
- **Windows**: 256x256 px minimum.
