# SOURCE OF TRUTH MAP

## Product Definition
Memoir.ai consolidates digital archives into a searchable encrypted timeline with AI narrative generation while preserving local-first data ownership.

## Core Flows
1. Create vault
2. Import archives
3. Normalize events
4. Search timeline
5. Generate narratives
6. Export data

## Architecture Components
Electron shell, renderer UI, worker processes, encrypted vault, ingestion pipelines, job runner, AI narrative engine, hybrid search.

## Security Guarantees
Local encrypted storage, no remote personal data storage, export and deletion control.

## AI Behavior
Narratives are citation-backed and versioned; outputs remain editable.
